TimeKeeper Help

Professional Race Timing with Video Review

Keyboard Shortcuts

TimeKeeper provides comprehensive keyboard shortcuts for efficient race timing operations:

Race Control

Shortcut Action Description
SPACE Record Video Start/stop video recording (most accessible key for critical recording)
ENTER Start/Stop Race Toggle race timing on/off
ESC Emergency Stop Immediately stop both race timing and video recording

Timeline Navigation (Video Review)

Shortcut Action Precision
← / → Navigate timeline ±10ms (standard)
⇧ + ← / → Fine navigation ±1ms (precise)
⌘ + ← / → Coarse navigation ±100ms (quick)
Home / End Jump to start/end Video boundaries

Finish Marking

Shortcut Action Requirement
M Add Marker Lane must be selected first (1-9)
Delete / ⌫ Remove last marker Removes most recent finish marker

Lane Management

Shortcut Action Description
1-9 Select Lane Choose active lane for marker placement
D Mark DNS Did Not Start (selected lane)
N Mark DNF Did Not Finish (selected lane)
S Mark DSQ Disqualified (selected lane)

Quick Actions

Shortcut Action Description
⌘ + E Export Frame Export current video frame as JPEG
⌘ + S Save Session Save current race timing data

Typical Race Timing Workflow

Setup Phase

  1. Click "NEW RACE" and enter race name and team/lane names
  2. Select camera from the dropdown menu
  3. Choose output folder for recordings and data

Race Timing

  1. Press ENTER to start race timing
  2. Press SPACE to begin video recording when competitors approach finish line
  3. Press ENTER to stop race timing when all competitors have finished
  4. Recording automatically stops when race ends

Video Review & Precise Timing

  1. Use 1-9 to select a lane
  2. Navigate video with ←/→ arrows
  3. Use ⇧ + arrows for frame-by-frame precision
  4. Press M to set finish marker at current position
  5. Repeat for each lane
  6. Press ⌘ + E to export finish line photo if needed
Note: You must select a lane (1-9) before adding a marker with the M key. The selected lane is shown in the "Selected Lane" indicator on the left panel.
Tip: For the most accurate timing, record only the finish line area. Start recording just before the first competitor arrives and stop after the last competitor finishes.

Lane Status Management

TimeKeeper supports standard race statuses:

  • Registered - Default status before race
  • Finished - Competitor completed the race (set by adding markers)
  • DNS - Did Not Start
  • DNF - Did Not Finish
  • DSQ - Disqualified

Video and Timing Synchronization

TimeKeeper automatically synchronizes video timestamps with race timing for accurate post-race analysis. The timeline view shows:

  • Race start and end markers
  • Video recording period
  • Individual finish markers for each lane
  • Precise timing down to millisecond accuracy

Data Export

All race data is automatically saved as JSON files alongside video recordings. You can:

  • Export individual finish line frames as high-quality JPEG images
  • Save complete race sessions with all timing data
  • Load previous sessions for review or correction
System Requirements: TimeKeeper requires macOS 13.0 (Ventura) or later and camera access permissions for video recording.
